Project

D-SIGN

About


     The goal of the project was to unite and teach designers about modern web practices. D-SIGN contained an instructional section full of introductory material to web design, and it had a browser based DOM Editor, which could help designers quickly explore layout designs without coding. This was entered into the Web Design Level 5 (Highest Level) Category at the ACTE Regional Tech Fair and again in the ACTE State Tech Fair, which it won First Place at both competitions. The site was comprised solely of HTML, CSS, and Javascript. The site was created in Feburary 2018 and the final design was finished by April 2018.
